## How to use 

```bash

# clone the repository
$ git clone https://github.com/j-rdel/desafio-supera.git

# Move to / of project
$ cd desafio-supera

# Install the dependencies
$ npm install

# Runs the app in the development mode.
$ npm start

# Open http://localhost:3000 to view it in the browser.

```
